# Turkish Finance AI Agent (Prototype)

This project is an experimental AI-based agent for Turkish financial markets.  
Goal is to collect market data and generate daily trading intelligence reports.

## Features
- BIST stock price fetching (yfinance)
- Basic price trend analysis
- Automatic markdown report generation
- Future: news sentiment & LLM commentary

## Tech Stack
- Python
- Pandas
- yfinance
- OpenAI API (planned)

## How to Run
```bash
pip install -r requirements.txt
python main.py
